# Glossary

## High Availability (HA)

Ability to keep service running through localized failures with minimal interruption.

## Disaster Recovery (DR)

Ability to recover service and data after a major outage, especially a regional event, based on defined RTO and RPO.

## Data Center Resiliency

Protection against a datacenter-level failure inside the same Azure region.

## Zonal Resiliency

Protection across Availability Zones within one Azure region.

## Regional Resiliency

Protection against loss or major degradation of an Azure region, usually involving cross-region replication and failover.

## RTO

Recovery time objective: the target time to restore a service after an interruption.

## RPO

Recovery point objective: the acceptable amount of data loss measured in time.

## Rule

Do not classify same-region zone redundancy as true DR unless Microsoft documentation explicitly does so.
